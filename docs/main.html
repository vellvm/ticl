  • ME.anr_qprog_bind_l: ANR for bind in queue programs.
  • ME.anr_pop_cons: ANR for pop in queue programs.
  • ME.aul_qprog_eventually: AUL for eventually in queue programs.
  • ME.aul_qprog_until_now: AUL for until_now in queue programs.
  • ME.aul_qprog_bind: AUL for bind in queue programs.
  • ME.aur_qprog_termination: AUR for termination in queue programs.
  • ME.aur_qprog_bind_r: AUR for bind in queue programs.
  • ME.aur_pop_cons: AUR for pop in queue programs.
  • ME.axr_qprog_ret: AXR for return in queue programs.
  • ME.axr_push: AXR for push in queue programs.
  • ME.axr_pop_nil: AXR for pop_nil in queue programs.
  • ME.cdenote_prog: Denotation of queue programs.
  • ME.encode_queueE: Encoding for queue events.
  • ME.equivr_ifsome_some: Equivalence for if-some.
  • ME.instr_prog: Instruction for programs.
  • ME.pop: Pop operation.
  • ME.push: Push operation.

  • ME.aur_sprog_bind_r: AUR for bind in store programs.
  • ME.aur_cprog_bind_r_eq: AUR for composed programs (equality).
  • ME.aur_cprog_bind_r: AUR for composed programs.
  • ME.denote_sprog: Denotation of store programs.
  • ME.denote_cprog: Denotation of composed programs.
  • ME.encode_secE: Encoding for security events.
  • ME.instr_sprog: Instruction for store programs.
  • ME.instr_cprog: Instruction for composed programs.
  • ME.ticll_sprog_if: Left Ticl for if in store programs.
  • ME.ticlr_cprog_if: Right Ticl for if in composed programs.
  • ME.ticlr_sprog_call: Right Ticl for call in store programs.
